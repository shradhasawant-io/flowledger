# FlowLedger - Architecture Decision Records (ADR)

This directory contains all major architectural decisions made during the development of FlowLedger.

Each ADR documents:

- The problem
- The decision
- Alternatives considered
- Why the decision was chosen
- Consequences

These records serve as the project's long-term architectural memory.
